## KeX

[![](https://jitpack.io/v/Keyrillanskiy/keX.svg)](https://jitpack.io/#Keyrillanskiy/keX)

**KeX** - **ke**yrillanskiy's e**x**tensions for kotlin and android. The goal of KeX is to store the most useful extensions in one library.

## How to
To get a Git project into your build:

Step 1. Add the JitPack repository to your build file

Add it in your root build.gradle at the end of repositories:

	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
Step 2. Add the dependency

	dependencies {
	        implementation 'com.github.Keyrillanskiy:keX:0.1'
	}
